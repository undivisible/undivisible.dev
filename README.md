# undivisible-next

Next.js source for [undivisible.dev](https://undivisible.dev): a static export (`output: "export"`) with a full-screen client shell, shader background, and an `/about` narrative with hover tooltips on project links.

## Why this repo is interesting

- **Dual delivery**: the interactive experience lives in Next.js (`components/SiteShell.tsx`, Framer Motion, Paper shaders). A **second artifact** is a single-file static landing generated by **Crepuscularity’s `crepus web build`** from `crepuscularity-site/site.json`—the same toolchain described on [semitechnological/crepuscularity](https://github.com/semitechnological/crepuscularity) (branch `web-v3`). That HTML is written to **`dist/index.html`** so it does not overwrite Next’s static export in `out/`. GitHub Pages deploys **`dist/`** only.
- **Honest “world’s first” claims**: the about copy aligns with Crepuscularity’s documented positioning (first GPUI-native stack with hot-reloaded `.crepus` templates; first plug-and-play Rust path to Manifest V3 extensions). Everything else is described by what it does, not hype.

## Commands

```bash
npm install
npm run dev          # local Next dev server
npm run build        # Next static export → ./out
npm run lint
```

**Static landing (Crepuscularity `crepus web`):** requires Rust **stable** (1.94+), Git, and Node for the link post-step.

```bash
npm run build:static
# writes ./dist/index.html (clones semitechnological/crepuscularity @ web-v3 under /tmp if needed)
```

## GitHub Actions

`.github/workflows/deploy.yml` clones `semitechnological/crepuscularity` at `web-v3`, builds `crepus` with `--no-default-features` (Linux-friendly, no GPUI desktop), runs `crepus web build` on `crepuscularity-site/`, then `scripts/apply-crepus-link-placeholders.mjs` (tokens in `site.json` become real anchors because the builder escapes HTML in JSON strings).
